TWM Library
-----------

TWM is a C++ library to act as a framework for students experimenting with
animation and effects programming.  The library is primarily targeted at
students studying the Animation and Simulation Programming module at
Teesside University.  It is constantly in development; you should check
regularly for new builds.

Documentation is provided via the School Wiki.  You are encouraged to add
to this documentation yourself as you learn to use the library.
